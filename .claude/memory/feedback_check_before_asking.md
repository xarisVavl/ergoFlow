Always read the relevant file before telling the user to make a change. The user finds it frustrating when asked to do something they may have already done.

**Why:** User explicitly asked for this behavior — "πριν μου πεις να κανω κατι θα κοιτας αν το εκανα".

**How to apply:** Before suggesting any edit or change, read the file first to verify the current state.
